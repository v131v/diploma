# Рубрика оценки архитектурных вариантов

Итоговый score: `0-100`

- `25` — архитектурная ясность и целостность
- `20` — качество data flow и transitions
- `20` — сила и корректность опоры на конспекты
- `15` — реализуемость и инженерная правдоподобность
- `10` — соответствие теме диплома
- `10` — novelty / полезная оригинальность без фантазирования

Tie-break:

1. меньше `critical + major`
2. выше балл за опору на конспекты
3. выше балл за реализуемость
4. меньший номер `design_id`
